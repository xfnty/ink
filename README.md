Requirements for building:
- [.NET 6.0 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/6.0)
- [Build Tools for Visual Studio](https://visualstudio.microsoft.com/downloads#build-tools-for-visual-studio-2026)
- `MSVC Build Tools for x64/x86`, `Windows 10/11 SDK` Visual Studio components

To build everything and start Windows Ink tester app, run `project build test`.

To package the plugin, run `project build dist`.
